Street Penalty Soccer - Multiplayer Game

DESCRIPTION

A Multiplayer game in which players try to score first 10 goals. This game has both single player and multiplayer feature. Its a turn based game, one player can shoot or defend at one time.

INSTALL NOW

FEATURES

* Multiplayer
* Score Card
* 2D Graphics

RULES

Tap appropriate side to position player
Tap appropriate side to shoot ball
Tap to jump/Defend
Score 10 goals to win

CONTROLS

*Touch Control

This game is made with Unity 5 Personal Edition. Please write me about any problem or suggestion.
