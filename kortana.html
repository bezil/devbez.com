Kortana Rally Driver - PC Game

DESCRIPTION

Kortana Rally Driver is a rally racing game in which player has to read out pace notes(Commands) for Driver to get best time records.

In this game Cortana is your Rally Driver, Player is the Pace note reader. If player reads pace note at the correct time, car speed increases. If player misses out or mispronounces pace note, car speed decreases. Objective of game is to finish 2 laps with best time record.

Requirements
*Microphone
*Cortana

Features
*Speech Recognition
*Open Game with Cortana

INSTALL NOW

This game is made with Unity 2017 Personal Edition.
